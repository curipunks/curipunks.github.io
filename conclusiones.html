Conclusiones

Diseñar el sitio web para Delicias Dulces fue una experiencia enriquecedora. El proceso implicó entender la identidad visual de la empresa, investigar teoría del color y aprender a estructurar el contenido de forma clara.

Las principales dificultades incluyeron la falta de experiencia con HTML y la gestión de imágenes. Sin embargo, con práctica y recursos en línea, se logró construir un sitio atractivo.

Este proyecto puede escalarse en el futuro con funciones como carrito de compra, registro de usuarios o integración con redes sociales.
